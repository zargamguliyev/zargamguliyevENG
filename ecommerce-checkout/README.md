# E-commerce Checkout Optimization

## Project Overview

Business analysis project focused on improving the checkout process of an e-commerce platform.

The goal was to identify bottlenecks in the checkout flow and design a more efficient user journey.

---

## Problem

Users often abandon the checkout process before completing the purchase.

Key issues:

- long and complex checkout flow
- unnecessary form fields
- unclear order confirmation process

These problems negatively affect conversion rate.

---

## Business Goal

Increase checkout conversion rate and reduce cart abandonment.

---

## AS-IS Process

Current checkout flow:

1. User adds products to cart
2. Opens checkout page
3. Fills personal information
4. Selects delivery method
5. Selects payment method
6. Confirms order

Problems identified:

- duplicate information fields
- too many steps in checkout
- unclear error messages

---

## TO-BE Solution

Simplified checkout process:

- reduce number of steps
- group related fields
- introduce autofill for user data
- improve error validation
